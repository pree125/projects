# Module 2: Objective data

* Bias
    * Systematically skew data
* Sampling bias:
    * Only sample small sample set
* Observer bias:
    * Different people observe different things
* Interpretation bias:
    * Interpret things in a positive or negative way
* Confirmation bias:
    * Confirm pre-existing belief

* Good data:
    * Reliable
    * Original
    * Comprehensive
    * Current
    * Cited
    * Vetted public data sets
* Bad data:
    * Incomplete, bias
    * Not original
    * Missing infomation
    * Outdated
    * Not cited

* Open data:
    * Free access
    * Use and share
    * daata.gov
    * Reuse and distribution
    * Universal participation
    * Interoperatability
